隐私与免责声明

数据存储说明

  • 完全本地存储：本工具产生的所有数据（包括爬取到的评论 CSV 文件、历史记录、账号配置）均存储在你的本地计算机上，不会上传至任何服务器。
  • 账号信息：你的 Cookie 等账号信息存储在项目目录下的 config.json 文件中。请妥善保管此文件，切勿将其发送给他人或上传到 GitHub 等公开平台。
  • 历史记录：任务历史记录保存在 history.json 中，仅记录任务参数和统计数据，不存储完整 Cookie。
  • 日志文件：爬取日志保存在各任务输出目录的 logs/ 子目录中，仅包含过滤后的运行信息，不含完整 Cookie 内容。

Cookie 脱敏机制

  • 账号管理列表中，Cookie 仅显示前 8 个字符 + "…"，不显示完整内容。
  • 实时日志中不会输出完整 Cookie；如需调试，仅记录最多前 8 个字符。
  • 历史记录中仅保存账号名称，不保存 Cookie 本体。
  • 错误堆栈写入磁盘前，程序会自动将 Cookie 相关字段替换为"***"。

网络请求说明

  • 本工具仅向 bilibili.com 的公开 API 发起请求，不访问任何第三方服务器。
  • 请求使用你提供的账号 Cookie，以已登录用户身份访问 API，与正常浏览器访问行为一致。
  • 本工具不包含广告、追踪代码或数据收集功能。
