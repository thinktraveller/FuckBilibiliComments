Cookie 获取（Chrome / Edge）

本章介绍在 Google Chrome 或 Microsoft Edge 浏览器中获取 B 站 Cookie 的方法。两款浏览器步骤完全相同。

方法一：使用 Cookie-Editor 扩展（推荐）

1：安装扩展

在浏览器地址栏输入 chrome://extensions/（Edge 用 edge://extensions/），打开扩展管理页面，点击右上角"开发者模式"，或直接在 Chrome 网上应用店搜索 Cookie-Editor 安装。

2：登录 B 站

访问 www.bilibili.com，用账号密码或扫码登录。

3：打开 Cookie-Editor

登录后，点击浏览器右上角扩展图标栏中的 Cookie-Editor 图标（形如一块饼干）。如果找不到，先点击拼图图标展开扩展列表。

4：导出 Cookie

在弹出面板底部，找到 Export（导出）按钮并点击，所有 Cookie 会以 JSON 格式自动复制到剪贴板。

5：粘贴到本程序

回到本程序，切换到 账号管理 Tab，点击 新增，在 Cookie 输入框中按 Ctrl+V 粘贴，填写账号名称后保存。

方法二：手动复制 SESSDATA（高级）

  1. 登录 B 站后，按 F12 打开开发者工具。
  2. 切换到 Application（应用） 标签页。
  3. 在左侧找到 Storage > Cookies > https://www.bilibili.com。
  4. 在右侧列表中找到名为 SESSDATA 的条目，双击其 Value 列，复制完整值。
  5. 将值粘贴到本程序账号管理的 Cookie 输入框（格式：SESSDATA=你复制的值）。

安全提示

Cookie 相当于你的登录凭证，请勿分享给任何人。本程序的 Cookie 仅存储在本地的 config.json 文件中，不会上传到任何服务器。
