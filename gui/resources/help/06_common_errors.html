常见错误与解决方法

Cookie 失效 / 被封禁

现象：日志显示"Cookie 无效"、"请求被拒绝"或爬取到 0 条评论；账号管理中测试有效性返回失败。

原因：B 站 Cookie 有效期通常为一个月；频繁爬取可能触发风控导致 Cookie 提前失效。

解决：重新登录 B 站后再次导出 Cookie 并更新账号信息；若多个账号均失效，等待 24 小时后再试。

触发限流 / 请求间隔过短

现象：日志显示"触发限流，等待"或"Rate limit"，爬取速度明显降低或中断。

原因：请求频率过高，被 B 站服务器限速。

解决：将"请求间隔"参数调大（建议 2~5 秒）；减少同时运行的爬取任务数量；换一个账号继续爬取。

视频不存在或已删除

现象：点击"解析"后显示"视频不存在"或"该视频已被删除"。

原因：BV 号输入有误，或视频已被 UP 主删除/B 站下架。

解决：检查输入的 BV 号是否完整（应以 BV 开头，后跟 10 个字母数字）；在浏览器中直接访问该 BV 号确认视频是否存在。

解析失败 / 网络错误

现象：点击"解析"后长时间无响应，或日志提示"网络连接失败"、"ConnectionError"。

原因：本机网络连接问题，或 B 站 API 临时故障。

解决：检查网络是否正常（可在浏览器中访问 bilibili.com 验证）；稍等几分钟后重试；如果使用了代理，确认代理配置正确。

爬取结果文件为空 / 0 条评论

现象：爬取完成，但输出 CSV 文件内容只有表头，没有数据。

原因：该视频评论区已关闭；Cookie 失效；或视频本身没有评论。

解决：先在浏览器中确认视频评论区是否可见；检查账号 Cookie 有效性；若评论区确实存在但爬取为空，可能触发了临时限流，等待后重试。

输出目录无权限

现象：日志显示"PermissionError"或"无法创建目录"。

解决：将输出目录切换到用户目录下（如 C:\Users\你的用户名\Documents\），避免使用 C 盘根目录或 Program Files 等受保护路径。

通用排查步骤

1. 查看日志窗口中带有 [ERROR] 或 [WARN] 标签的行，通常包含错误原因。
2. 在"历史记录"Tab 中找到该任务，查看"错误信息"字段。
3. 如果问题持续，尝试重新导出 Cookie 并更新账号。
