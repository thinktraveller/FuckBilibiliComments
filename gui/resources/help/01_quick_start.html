快速上手

按照以下步骤，可在 5 分钟内完成第一次评论爬取。

第一部分：准备工作

步骤一：安装 Cookie-Editor 扩展

访问 Cookie-Editor 官方网站 https://cookie-editor.com/，根据你的浏览器选择对应的下载链接。

安装成功后，浏览器右上角工具栏会出现 Cookie-Editor 的图标。

注意

请务必通过官网链接进入对应商店安装，不要在扩展商店中自行搜索，搜索结果中存在仿冒插件，可能造成账号信息泄露。

步骤二：导出 B 站 Cookie

访问 bilibili.com 并登录你的账号，然后：

  1. 点击浏览器右上角的 Cookie-Editor 图标
  2. 在弹出面板底部点击 Export（导出）
  3. Cookie 字符串将自动复制到剪贴板

步骤三：获取 User-Agent

User-Agent 用于告诉 B 站"请求来自哪种浏览器"，请填写你实际使用的浏览器的 UA。可通过以下方式查找：

  • Chrome：地址栏输入 chrome://version，找到"用户代理"
  • Edge：地址栏输入 edge://version，找到"用户代理"
  • Firefox：地址栏输入 about:support，找到"用户代理"
  • 通用方法：F12 → Network → 刷新页面 → 点击任意请求 → Request Headers → User-Agent

步骤四：在账号管理中录入账号信息

切换到 账号管理 Tab：

  1. 点击 新增
  2. 填写账号名称（自定义备注，方便区分）
  3. 在 Cookie 输入框中粘贴刚才复制的内容
  4. 粘贴你的 User-Agent
  5. 点击 保存，再点击 设为当前账号

安全提醒

Cookie 是你 B 站账号的登录凭证，请勿分享给他人，也不要将包含 Cookie 的配置文件上传到任何平台。

第二部分：开始爬取

步骤五：切换到 评论爬取 Tab

在顶部输入框粘贴 B 站视频链接（如 https://www.bilibili.com/video/BV1xx...）或 BV 号，点击 解析，确认视频信息无误后，点击 开始爬取。

提示

首次爬取建议选择默认的 综合模式，覆盖率最高。爬取过程中可随时切换到其他 Tab，程序会在后台继续运行。

第三部分：查看结果

爬取完成后会弹出提示框，点击 打开输出文件夹 可直接查看生成的 CSV 文件。也可以在 历史记录 Tab 中找到所有过往任务，双击行可直接打开对应输出文件夹。
