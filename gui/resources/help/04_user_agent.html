User-Agent 获取

User-Agent（UA）是浏览器向服务器报告自身信息的字符串，使用与你浏览器一致的 UA 可以降低被 B 站限流的概率。

提示

本程序账号管理中内置了常用 UA 预设（Firefox / Chrome），大多数情况下直接选择预设即可，无需手动获取。只有当默认 UA 被频繁限流时才建议自定义。

从浏览器开发者工具获取

1：打开开发者工具

在任意页面按 F12，或右键页面空白处选择"检查"，打开浏览器开发者工具。

2：切换到 Network（网络）标签页

在开发者工具顶部找到并点击 Network（中文版为"网络"）标签。如果看不到，可能需要先点击 >> 展开更多标签。

3：刷新页面，抓取请求

按 F5 或 Ctrl+R 刷新页面，此时 Network 面板会出现大量网络请求。

4：找到 User-Agent

点击列表中任意一个请求（通常第一个文档类型请求即可），在右侧 Headers（标头） 面板中向下滚动，找到 Request Headers 区域，其中 User-Agent 对应的值即为你的 UA。

5：复制并填入本程序

右键点击 User-Agent 的值，选择"复制值"，然后在本程序账号管理中选择 自定义，粘贴到 UA 输入框中保存。

常见 UA 示例

Chrome 124（Windows）：

Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/124.0.0.0 Safari/537.36

Firefox 125（Windows）：

Mozilla/5.0 (Windows NT 10.0; Win64; x64; rv:125.0) Gecko/20100101 Firefox/125.0

Edge 124（Windows）：

Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/124.0.0.0 Safari/537.36 Edg/124.0.0.0
